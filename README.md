# XIONG WENRUI — Portfolio

个人作品集网站，赛博风格，纯静态。

**Live:** https://xiongwenrui.cn

## Stack

- HTML + CSS + Vanilla JS（零依赖）
- 部署：Vercel（自动 CI/CD）
- 域名：xiongwenrui.cn（阿里云）

## Development

```bash
python3 -m http.server 8848
# → http://localhost:8848
```

## Structure

- `index.html` — 主页面
- `style.css` — 全部样式（赛博风 + 电光青单色）
- `script.js` — 交互（打字机、滚动、彩蛋）

## 说明

作品视频通过百度网盘链接查看，网站展示封面图 + 项目信息。
